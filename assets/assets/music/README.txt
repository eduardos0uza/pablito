Coloque aqui sua música fixa para o site.

Como usar:
- Copie seu arquivo MP3/OGG/WAV para esta pasta.
- No arquivo index.html, o elemento <audio id="audio"> usa o atributo data-src.
- Ajuste o caminho, por exemplo: data-src="assets/music/sua-musica.mp3".

Notas:
- Recomendo MP3 a 128kbps para tamanho moderado.
- Se quiser autoplay funcionando em todos navegadores, mantenha o volume inicial baixo.

Esta pasta existe para facilitar o deploy da música fixa.
